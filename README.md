# Machine Learning Projects

This repository contains machine learning projects developed using Python and popular machine learning libraries. The projects demonstrate data preprocessing, exploratory data analysis, model building, evaluation, and performance comparison using supervised learning techniques.

## Projects

### 1. API Fraud Detection Using Machine Learning

**Objective:**
Detect fraudulent API requests and classify API activities using machine learning algorithms.

**Algorithms Used**
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- One-Class SVM
- Isolation Forest
- Local Outlier Factor

**Technologies**
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

---

### 2. YouTube View Prediction Using Machine Learning

**Objective:**
Predict YouTube video views using regression models based on user engagement metrics.

**Algorithms Used**
- Multiple Linear Regression
- Decision Tree Regression
- Random Forest Regression
- XGBoost Regression

**Technologies**
- Python
- Pandas
- NumPy
- Scikit-learn
- Statsmodels
- XGBoost
- Matplotlib
- Seaborn
