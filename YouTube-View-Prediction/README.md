# YouTube View Prediction Using Machine Learning

## Project Overview

This project predicts YouTube video views using machine learning regression models based on engagement metrics such as likes and comments.

## Dataset

The dataset contains YouTube video statistics collected from Data Science channels.

## Features

- Like_count
- Comment_Count

## Target Variable

- Views

## Machine Learning Models

- Multiple Linear Regression
- Decision Tree Regression
- Random Forest Regression
- XGBoost Regression

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Statsmodels
- XGBoost
- Matplotlib
- Seaborn

## Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Selection
5. Regression Modeling
6. Model Evaluation
7. Performance Comparison

## Evaluation Metrics

- R² Score
- RMSE
- MAE
- MSE

## Best Performing Model

Based on the evaluation results, **Multiple Linear Regression** achieved the highest predictive accuracy on the test dataset.

## Conclusion

The developed model can help estimate YouTube video views based on engagement metrics and support data-driven content analysis.
